# React and Angular single-spa example

## Running the app
#### Clone the code 
```bash
git clone  && cd react-angular-single-spa
```

#### Install dependencies
```bash
npm install
```

#### Run app
```bash
npm start
```

#### Open `http://localhost:9090/#/` in browser

You should see something like this:
